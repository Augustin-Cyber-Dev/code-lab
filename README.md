## 📖 Sobre o Projeto
 
O **Code Lab** é um site institucional criado para simular a presença digital de uma empresa fictícia de desenvolvimento de software. O projeto foi desenvolvido como exercício prático para aplicação de conceitos de desenvolvimento web frontend, com foco em layout responsivo, organização de conteúdo e experiência do usuário.
 
---
 
## ✨ Funcionalidades
 
- 🏠 Página inicial com apresentação da empresa e seus diferenciais
- 💼 Seção de serviços: desenvolvimento web, mobile e sistemas
- 👥 Página dedicada à equipe de desenvolvedores
- 📬 Formulário de contato funcional
- 📱 Layout totalmente responsivo para todos os dispositivos
- 🎨 Design moderno e profissional
